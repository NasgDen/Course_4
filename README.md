# Проект домашнего задания

## Описание

Домашняя работа по созданию классов. В проекте создается 2 класса: Category и Product.

## Описание функций

1. Модуль category.py - создается class Category со следующими свойствами:
* name (название),
* description (описание)
* price (цена)
* quantity (количество в наличии)
2. Модуль product.py - создается class Product со следующими свойствами:
* name (название)
* description (описание)
* products (список товаров категории)
и атрибутами класса:
* category_count(количество категорий)
* product_count (количество товаров)
* 
## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/NasgDen/Course_4
```
2. В PyCharm установите менеджер проекта poetry.

## Использование:

1. Запустите файл main.py из корневой директории проекта.
2. Просмотрите результат выполнения программы.

## Тестирование модулей

Проводилось тестирование все имеющихся модулей с помощью фрейворка pytest и метрики Code coverage.
Все тесты находятся в папке test.
Результаты покрытия тестов находятся в папку htmlcov.
