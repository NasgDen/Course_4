# Проект домашнего задания

## Описание

Домашняя работа по созданию классов. В проекте создается 2 класса: Category и Product.

## Описание функций

1. Модуль category.py - создается class Category со следующими свойствами:
* name (название),
* description (описание)
* price (цена)
* quantity (количество в наличии)
2. Модуль product.py - создается class Product со следующими свойствами:
* name (название)
* description (описание)
* products (список товаров категории)
и атрибутами класса:
* category_count(количество категорий)
* product_count (количество товаров)
3. Модуль smartphone - создан дочерний класс Smartphone от класса Product, расширен атрибутами:
* efficiency (производительность)
* model (модель)
* memory (объем встроенной памяти)
* color (цвет)
4. Модуль lawngrass - создан дочерний класс LawnGrass от класса Product, расширен атрибутами:
* country (страна-производитель)
* germination_period (срок прорастания)
* color (цвет)
5. Модуль Base_class - создан абстрактный класс для класса Product
6. Модуль mixin_print - создан класс-миксин, который при создании объекта распечатывает в консоль информацию о том, от какого класса и с какими параметрами был создан объект.
7. Модуль abstract - создан абстрактный класс для класса Category и Order

## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/NasgDen/Course_4
```
2. В PyCharm установите менеджер проекта poetry.

## Использование:

1. Запустите файл main.py из корневой директории проекта.
2. Просмотрите результат выполнения программы.

## Тестирование модулей

Проводилось тестирование все имеющихся модулей с помощью фрейворка pytest и метрики Code coverage.
Все тесты находятся в папке test.
Результаты покрытия тестов находятся в папку htmlcov.
